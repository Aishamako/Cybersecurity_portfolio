## Featured Project: Simplified Predictive Risk Reporting Tool
A lightweight AI-augmented web application designed to help users identify and report cyber threats more efficiently. The tool uses a custom-trained spaCy NER model to extract cybersecurity entities from text and supports real-time enrichment using VirusTotal and Shodan.

**Key Features**
- Custom spaCy Named Entity Recognition (NER) model for cyber threat entity tagging (e.g., vulnerabilities, actors, attack types)
- Real-time threat intelligence enrichment via VirusTotal and Shodan APIs
- Secure authentication with password hashing and Email-based 2FA (Mailtrap for testing)
- Automated report generation with export to TXT and PDF (ReportLab)
- Dashboard UI with visualisation, dark mode, and report history

**Tech Stack**
Python, Flask, spaCy, SQLite, HTML/CSS/Bootstrap, REST APIs, ReportLab, Git/GitHub
